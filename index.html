📖 How to Use

  1. Search: Type a Pokémon name. It will automatically target the base form (e.g., searching Walrein targets Spheal).
  2. League: Choose your target CP limit (LL, GL, or UL).
  3. Options:
    • Include Evolutions: Adds the whole family to the filter.
    • Only Shiny: If checked, the bot will only catch the shiny version.
  4. Download: Add as many Pokémon as you want, then hit the green download button.
  5. PolygonX: Import the file in your app's Encounter Filters section.
